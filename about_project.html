O projekcie

Polecenie

Opracować i zaimplementować algorytm poszukujący maksymalnego zbioru krawędzi niezależnych grafu spójnego.

Rozwiązanie

graf spójny - graf w którym dla każdej pary wierzchołków istnieje łącząca je ścieżka

zbiór krawędzi niezależnych - inaczej skojarzenie

skojarzenie maksymalne - jest to takie skojarzenie w grafie , że po dodaniu dowolnej krawędzi spośród krawędzi G do tego skojarzenia, przestaje ono być skojarzeniem.

Algorytm

Model algorytmu w pseudokodzie:

maksymalne_skojarzenie(graf){
  lista_wierzchołków = E(graf)  // lista_wierzchołków będzie zbiorem wierzchołków niezawartych w skojarzeniu
  skojarzenie = []
  
  licznik = 0
  dopóki (długość(lista_wierzchołków) > 1){
    W = lista_wierzchołków.pop()
    znaleziono = Flase
    // szukamy nowych krawędzi dla skojarzenia
    dla każdego W2 w lista_wierzchołków {
      jeśli W-W2 należy do E(graf){
        skoajrzenie.dodaj(W-W2)
        lista_wierzchołków.usuń(W)
        lista_wierzchołków.usuń(W2)
        znaleziono = True
        licznik = 0
        break
      }
    }
    jeśli not znaleziono{
      lista_wierzchołków.usuń(W)
      lista_wierzchołków.dodaj_na_koniec(W)
      licznik ++
    }
    // czy minął pełen obrót wirzchołków bez znalezienia nowych krawędzi do skojarzenia?
    jeśli licznik > długość(lista_wierzchołków){
      break
    }
  }

  return skojarzenie
}

Dowód poprawności

Algorytm dodaje krawędzie do skojarzenia dopóki jest to możliwe, gdy przestaje być to możliwe to zgodnie z definicją skojarzenia maksymalnego otrzymujemy takie skojarzenie.

Algorym zatzymuje się po wykorzystaniu wszystkich wierzchołków lub po bezowocnym pełnym cyklu sprawdzania wierzchołków - czy mogą utworzyć nową krawędź w skojarzeniu.

Złożoność obliczeniowa

Pesymistyczna złożoność obliczeniowa: O(V*V)
